# دليل استخدام التقويم الأسبوعي - التصميم الأفقي

## 📅 نظرة عامة

التقويم الأسبوعي الجديد مصمم بنفس نمط Google Calendar التقليدي:
- **الأيام**: تعرض بشكل أفقي (من اليمين لليسار)
- **الأوقات**: تعرض بشكل عمودي تحت كل يوم
- **الخلايا**: تعرض حالة كل فترة (متاح/محجوز/غير متاح)

### الألوان

- ✅ **أخضر** (#d4edda): فترة متاحة للحجز
- ❌ **أحمر** (#f8d7da): فترة محجوزة
- ⭕ **رمادي** (#e9ecef): فترة خارج أوقات العمل
- 🔵 **أزرق** (#0d6efd): الفترة المحددة حالياً
- ⭐ **تمييز اليوم الحالي**: حدود زرقاء على العمود

## 📱 الميزات

### 1. التفاعل
- **Hover**: تكبير خفيف وتغيير لون عند المرور
- **Selection**: تمييز واضح للفترة المحددة
- **Tooltip**: عرض معلومات عند المرور على الخلية

### 2. التجاوب
- **الشاشات الكبيرة**: عرض كامل مع مساحات مريحة
- **الشاشات المتوسطة**: تقليل الأحجام مع الحفاظ على الوضوح
- **الشاشات الصغيرة**: تحسين للقراءة واللمس

### 3. الميزات الإضافية
- **تمييز اليوم الحالي**: حدود زرقاء على عمود اليوم
- **حالات Loading**: مؤشر تحميل واضح
- **حالة فارغة**: رسالة وديّة عند عدم وجود فترات

### تعديل الألوان

في ملف `weekly-calendar-horizontal.css`:

```css
.weekly-calendar-slot-cell.available {
    background-color: #d4edda;
    color: #155724;
}

.weekly-calendar-slot-cell.busy {
    background-color: #f8d7da;
    color: #721c24;
}
```

## 🔍 استكشاف الأخطاء

### مشاكل شائعة

1. **التقويم لا يظهر**:
   - تأكد من تحميل ملفات CSS
   - تحقق من وجود عنصر `weeklyCalendarGrid`
   - راجع console للأخطاء

2. **الفترات لا تظهر**:
   - تأكد من تحميل البيانات من Firebase
   - تحقق من إعدادات أوقات العمل
   - راجع availability slots

3. **الحجز لا يعمل**:
   - تأكد من تفعيل selectSlot()
   - تحقق من Firebase permissions
   - راجع booking logic
