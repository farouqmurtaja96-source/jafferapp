# تصميم التقويم الأسبوعي الجديد

## 📅 نظرة عامة على التصميم

تم تصميم التقويم الأسبوعي الجديد ليكون مشابهاً لتقويم Google Calendar وPreply، مع تحسينات خاصة لتطبيق تعلم اللغة العربية.

## 🎨 المميزات الرئيسية

### 1. التخطيط العام
- **العمود الأيسر**: يعرض الأيام والتواريخ
  - اسم اليوم بالعربية (السبت، الأحد، إلخ)
  - التاريخ (اليوم/الشهر)
  - ثابت عند التمرير الأفقي

- **الصف العلوي**: يعرض فترات الوقت
  - فترات 30 دقيقة
  - من 09:00 إلى 20:30
  - ثابت عند التمرير العمودي

- **الخلايا**: تعرض حالة كل فترة
  - ✅ متاح (أخضر)
  - ❌ محجوز (أحمر)
  - ⭕ غير متاح (رمادي)

### 2. الألوان
- **الأخضر** (#d4edda): الفترات المتاحة للحجز
- **الأحمر** (#f8d7da): الفترات المحجوزة
- **الرمادي** (#e9ecef): الفترات خارج أوقات العمل
- **الأزرق** (#0d6efd): الفترة المحددة حالياً

### 3. التفاعل
- **Hover Effects**:
  - تكبير الخلية قليلاً
  - تغيير اللون
  - ظل خفيف

- **Selection**:
  - تمييز واضح للفترة المحددة
  - ظل بارز
  - تكبير ملحوظ

### 4. التجاوب (Responsive)
- **الشاشات الكبيرة** (>768px):
  - عرض كامل للفترات
  - أحجام خلايا مريحة

- **الشاشات المتوسطة** (480px - 768px):
  - تقليل أحجام الخلايا
  - تحسين المسافات

- **الشاشات الصغيرة** (<480px):
  - تكبير الفترات
  - تحسين القراءة

## 📱 الميزات الإضافية

### 1. Tooltips
- عرض معلومات عند hover
- حالة الفترة (متاح/محجوز/غير متاح)
- الوقت المحدد

### 2. Loading State
- مؤشر تحميل واضح
- رسالة وديّة
- تصميم احترافي

### 3. Empty State
- رسالة واضحة عند عدم وجود فترات
- أيقونة توضيحية
- توجيه للمستخدم

## 🎯 التخصيص

### تعديل الأوقات
يمكن تعديل مصفوفة `TIME_SLOTS` في ملف `weekly-calendar.js`:

```javascript
const TIME_SLOTS = [
    '09:00', '09:30', '10:00', '10:30', '11:00', '11:30',
    '12:00', '12:30', '13:00', '13:30', '14:00', '14:30',
    '15:00', '15:30', '16:00', '16:30', '17:00', '17:30',
    '18:00', '18:30', '19:00', '19:30', '20:00', '20:30'
];
```

### تعديل الألوان
يمكن تعديل الألوان في ملف `weekly-calendar-new.css`:

```css
.weekly-calendar-slot-cell.available {
    background-color: #d4edda;
    color: #155724;
}

.weekly-calendar-slot-cell.busy {
    background-color: #f8d7da;
    color: #721c24;
}
```
